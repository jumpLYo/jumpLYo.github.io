SQL数据库操作命令大全

登陆数据库

mysql u 用户名 -p 密码

高频指令

  • 查看数据库
    show databases
  • 切换数据库
    use dbname
  • 查看表
    show tables
  • 查看表项结构
    desc

数据库授权

mysql> grant all privileges on *.* to 'abc'@'%' identified by '123456' with grant option ;
  • all privileges :表示将所有权限授予给用户，也可指定具体的权限，如：select 、create 、drop 等
  • on :表示这些权限对哪些数据库和表生效，格式：数据库.表名，这里写“*”表示所有的数据库、所有的表。如果我要指定将权限应用到test库的user表中，可以这么写:test.user
  • to ：将权限授予哪个用户。格式:”用户名”@”登录IP或域名”。%表示没有限制，在任何机器中都可以登录。比如:”abc”@”192.168.0.%”,表示abc这个用户只能在192.168.0.* 这个IP段中登录
  • identified by : 指定用户的登录密码
  • with grant option :表示允许用户将自己的权限授权给其他用户
给表 user 设置字符集utf8_general_ci或其他字符集
alter table user convert to character set utf8 collate utf8_general_ci

一、 基础

1、创建数据库

create database database-name

2、删除数据库

drop database dbname

4、创建表

create table tabname(col1 type1 [not null] [primary key],col2 type2 [not null],..)

举个栗子，如下创建一张用户表：

CREATE TABLE USER(
     id INT NOT NULL AUTO_INCREMENT,
   username VARCHAR(255) NOT NULL,
   PASSWORD VARCHAR(255) NOT NULL,
   primary KEY id  (id)
 ) ENGINE=INNODB DEFAULT CHARSET=utf8

5、删除表

drop table tabname

6、为表添加一列

alter table tabname add column col type

举个栗子,添加一列生日字段：

ALTER TABLE USER ADD COLUMN  birthday DATE DEFAULT NULL ;

7、添加主键 与 删除主键

  1. 删除主键(有些数据库删除会出错)
alter table tabname drop primary key(col)
  1. 添加主键
    alter table tabname add primary key(col)

8、创建索引(可以加快数据查询的速度) 与 删除索引

create [unique] index idxname on tabname(col...)

举个栗子，为表中username创建普通索引

CREATE INDEX username ON USER(username)

9、创建视图 与 删除视图（视图可以理解数据更好的展示，联合多张表数据产生的虚拟表，这块暂不考虑）

  1. 创建视图
create view viewname as select statement
  1. 删除视图
drop view viewname

10、常用的基本的SQL 语句

  • 查询
    select * from table where [condition]
  • 保存数据（插入数据）
    insert into table(field1,feild2) values(value1,value2)
  • 删除
    delete from table where [condition]
  • 更新
    update table set field1=value1,set field2=value2 where [condition]
  • 模糊查询
    select * from table where field1 like %zhangsan%
  • 排序
    select * from table order by field1,field2 [desc/asc]
  • 总数
    select count as totalcount from table
  • 求和
    select sum(field1) as sumvalue from table
  • 求平均
    select avg(field1) as avgvalue from table
  • 最大
    select max(field1) as maxvalue from table
  • 最小
    select min(field1) as minvalue from table

二、 真实操作

1、 创建person表

CREATE TABLE person(
    id_p INT NOT NULL ,
    lastName VARCHAR(64) NOT NULL,
    firstName VARCHAR(64) NOT NULL,
    address VARCHAR(255) DEFAULT NULL,
    city VARCHAR(64) DEFAULT NULL,
    PRIMARY KEY(id_p)
 )ENGINE=INNODB DEFAULT CHARSET=utf8

导入person数据

2、创建订单表

CREATE TABLE orders(
   id_o INT NOT NULL,
   orderNo VARCHAR(32) NOT NULL,
   id_p INT NOT NULL,
   PRIMARY KEY(id_o)
 ) ENGINE=INNODB DEFAULT CHARSET=utf8

3、基于以上两张表学习，常用的Sql操作

  1. 连接（join）
SELECT * FROM person AS p JOIN orders AS o
 ON p.id_p=o.id_o
 ORDER BY p.lastName

这种方式不会产生中间表，性能较好

SELECT * FROM person AS p ,orders AS o
 WHERE p.id_p=o.id_o
  1. 内连接(inner join) 与 JOIN 是相同的
SELECT  * FROM person AS p INNER JOIN orders AS o
 ON p.id_p=o.id_o
 ORDER BY p.lastName
  1. 左连接（left join）,以左表为主，左边表数据全部展示，右表匹配的数据也会展示
SELECT * FROM person AS p LEFT JOIN orders AS o
 ON p.id_p=o.id_p
 ORDER BY p.lastName
  1. 右连接(right join),以右表为主，匹配的行会展示，右表的数据都会展示
SELECT * FROM person AS p RIGHT JOIN orders AS o
 ON p.id_p=o.id_p
 ORDER BY p.lastName
  1. 全连接(full join)，只要参与连接的表中，存在匹配，full join 关键字就会返回行。(mysql 不支持)
SELECT * FROM person AS p  FULL JOIN orders AS o
 ON p.id_p=o.id_p
 ORDER BY p.lastName
  1. union 用于合并两个或多个 SELECT 语句的结果集(UNION 内部的 SELECT 语句必须拥有相同数量的列。列也必须拥有相似的数据类型。同时，每条 SELECT 语句中的列的顺序必须相同)
SELECT * FROM person AS p1 WHERE p1.id_p=1
 UNION
 SELECT * FROM person AS p2 WHERE p2.id_p=2
 UNION
 SELECT * FROM person AS p3 WHERE p3.id_p=6
  1. 分组(group by)
SELECT p.lastName,SUM(o.price) AS totalPrice FROM person AS p             ,orders AS o
 WHERE p.id_p=o.id_p
 GROUP BY p.lastName
